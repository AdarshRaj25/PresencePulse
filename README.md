# PresencePulse

An Android app that detects **phubbing** — when someone is physically present with others but using their phone.

## How it works
- Uses **BLE (Bluetooth)** to detect nearby people
- Uses **UsageStats** to detect phone usage
- Combines both signals to calculate a real-time Presence Score

## Features
- Real-time phubbing detection
- App usage tracking with categories
- BLE device scanning for proximity detection
- Rule-based scoring engine

## Built With
- Kotlin
- Android Room Database
- BLE (Bluetooth Low Energy)
- UsageStats API
